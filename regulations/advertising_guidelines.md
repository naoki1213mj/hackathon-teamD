# 景品表示法 チェックリスト（旅行業向け）

## 1. 有利誤認表示の禁止

### NGパターン
- ❌「最安値」「どこよりも安い」→ 根拠なく価格の優位性を主張
- ❌「今だけ○○円」→ 実際には常時その価格で販売している場合
- ❌「通常価格□□円のところ○○円」→ 通常価格で販売した実績がない場合（二重価格表示）
- ❌「50%OFF」→ 割引の基準となる元値が不当な場合

### OKパターン
- ✅「お得な価格帯」「お手頃価格」→ 具体的な比較を避けた表現
- ✅「期間限定（3/1〜3/31）○○円」→ 具体的な期日を明記
- ✅「前年同時期比 ○○円お得」→ 比較対象と根拠を明示

## 2. 優良誤認表示の禁止

### NGパターン
- ❌「業界No.1」「日本一」→ 客観的な調査データの裏付けがない場合
- ❌「絶対に満足」「100%楽しめる」→ 断定的な品質保証
- ❌「完全プライベートビーチ」→ 事実と異なる場合
- ❌「ミシュラン三つ星レストラン」→ 実際には掲載されていない場合

### OKパターン
- ✅「多くのお客様にご好評いただいています」→ 一般的な評価の表現
- ✅「○○サイト評価 4.5/5.0（2025年3月時点）」→ 出典と時点を明記
- ✅「きっとご満足いただけます」→ 推量表現

## 3. 不当表示チェック項目

| チェック項目 | 確認内容 |
|-------------|---------|
| 価格表示 | 税込み・税抜きを明記しているか |
| 二重価格 | 元値での販売実績はあるか（過去8週間以内に相当期間販売） |
| 期間限定 | 具体的な開始日・終了日を明記しているか |
| 条件 | 適用条件（人数・日程等）を明記しているか |
| 写真 | イメージ写真の場合「※写真はイメージです」と表記しているか |
| 口コミ引用 | 出典と時点を明記しているか |
| 比較広告 | 比較対象と根拠を客観的に示しているか |

## 4. NG 表現一覧

| NG表現 | 理由 | 修正案 |
|--------|------|--------|
| 最安値 | 有利誤認のおそれ | お得な価格帯 |
| 業界No.1 | 優良誤認のおそれ | 多くのお客様に選ばれている |
| 絶対 | 断定的表現 | きっと / おそらく |
| 完全保証 | 有利誤認のおそれ | 充実のサポート体制 |
| 今だけ | 有利誤認（期間限定の根拠不足） | 期間限定（具体的な期日を明記） |
| 日本一 | 優良誤認のおそれ | 高い評価をいただいている |
| 永久 | 断定的表現 | 長期間 |
| 唯一 | 優良誤認のおそれ | 希少な / 珍しい |
| 激安 | 有利誤認のおそれ | リーズナブル / お手頃 |
| 限定○名 | 根拠のない限定数量 | 先着順（具体的な条件を明記） |
